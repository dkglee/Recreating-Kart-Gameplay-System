## 주요 기능

### 차량 물리
카트라이더 특유의 주행 감각을 언리얼 엔진 5에서 구현하기 위해, 다음의 물리 공식들을 사용하여 특유의 트리프트와 주행 감각을 구현하였습니다.

1. 가속 (주행 감각)
2. 서스펜션 (특유의 탄성력)
3. 스티어링 토크 (주행 감각)
4. 측면 마찰력 (드리프트 연출 및 부스터 게이지 등 연출)
